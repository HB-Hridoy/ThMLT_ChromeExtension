ThMLT Color Theme Editor

ThMLT is a Chrome extension that allows you to create and manage color themes for your projects. Easily import, export, and edit color themes with a user-friendly interface.

Learn more

Sample JSON Structure

      {
        "projectName": "My Project",
        "author": "Author Name",
        "version": "1.0.0",
        "colors": {
          "primitives": {
            "primary": "#FF5733",
            "secondary": "#33FF57"
          },
          "semantic": {
            "background": "primary",
            "text": "secondary"
          }
        }
      }
